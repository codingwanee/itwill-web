내장함수 - Timer 함수(시간 관련 함수)

setTimeout(callback,ms) : 시간(ms) 경과 후 callback 함수의 명령을 한 번만 실행하는 함수 - timeoutId 반환

clearTimeout(timeoutId) : timeoutId로 등록된 setTimeout() 함수를 초기화 시키는 함수 - callback 함수의 명령 취소

setInterval(callback,ms) : 시간(ms)마다 callback 함수의 명령을 반복 실행하는 함수 - intervalId 반환

clearInterval(intervalId) : intervalId로 등록된 setInterval() 함수를 초기화 시키는 함수 - callback 함수의 명령 취소
