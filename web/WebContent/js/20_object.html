자바스트립트 객체

객체 : 속성(Property : 값)과 기능(Method : 함수)의 모임

자바스트립트는 브라우저를 객체로 표현하여 프로그램 작성(BOM:Browser Object Model) - 객체 관점 프로그래밍

자바스크립트는 Object 클래스를 이용하여 객체를 생성하고 객체에 요소(프로퍼티 또는 메소드)를 추가,변경,삭제하여 사용

Object 클래스 : 요소가 선언되어 있지 않은 클래스

{} 기호를 이용하여 Object 객체 생성 가능 - JSON(JavaScript Object Notation : 자바스트립트 객체 표기법)
